**CompTIA Network+ (N10-009)**

## What is NFV?

**Network Function Virtualization (NFV)** is a technology that **replaces dedicated network hardware appliances with software-based network functions running on standard servers**.

## Advantages of NFV

* **Cost-efficient:** Reduces dependency on specialized hardware
* **Flexible & Scalable:** Deploy or scale network functions on-demand
* **Rapid Deployment:** New services can be deployed via software
* **Simplified Management:** Centralized orchestration of virtual network functions

---

## NFV vs Traditional Network Appliances

| Feature     | NFV            | Traditional Network Hardware   |
| ----------- | -------------- | ------------------------------ |
| Deployment  | Software-based | Dedicated hardware             |
| Flexibility | High, scalable | Low, hardware-limited          |
| Cost        | Lower CAPEX    | High CAPEX & OPEX              |
| Maintenance | Easier updates | Hardware replacements required |

---

## Common Use Cases

* Enterprise and ISP networks needing virtualized firewalls, routers, or load balancers
* Rapid service deployment in cloud and data centers
* Network service orchestration and automation

## Exam Tips

* Know **examples: vRouter, vFirewall, vLB, VPN services**
* Focus on **benefits: cost, scalability, flexibility**
