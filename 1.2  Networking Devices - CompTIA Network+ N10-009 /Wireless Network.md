## What is a Wireless Network?

A **wireless network** allows devices to communicate and access network resources **without physical cables**, using **radio frequency (RF) signals**.

## Wireless Standards (802.11 Family)

| Standard           | Frequency | Max Speed | Notes                               |
| ------------------ | --------- | --------- | ----------------------------------- |
| 802.11a            | 5 GHz     | 54 Mbps   | Less interference                   |
| 802.11b            | 2.4 GHz   | 11 Mbps   | Longer range, more interference     |
| 802.11g            | 2.4 GHz   | 54 Mbps   | Backward compatible with 802.11b    |
| 802.11n            | 2.4/5 GHz | 600 Mbps  | MIMO support                        |
| 802.11ac           | 5 GHz     | 1.3 Gbps+ | Wide channels, MIMO, beamforming    |
| 802.11ax (Wi-Fi 6) | 2.4/5 GHz | 10 Gbps   | OFDMA, MU-MIMO, improved efficiency |

ðŸ“Œ Remember **frequency, speed, and key features**

## Wireless Security Protocols

| Protocol | Security Level | Notes                                               |
| -------- | -------------- | --------------------------------------------------- |
| WEP      | Weak           | Deprecated, easy to crack                           |
| WPA      | Moderate       | Uses TKIP, better than WEP                          |
| WPA2     | Strong         | AES encryption, widely used                         |
| WPA3     | Very Strong    | Enhanced security, resistant to brute-force attacks |

## Common Wireless Network Issues

* Interference from microwaves, cordless phones
* Weak signal / coverage issues
* SSID not broadcasting
* Wrong security configuration
* Rogue APs (unauthorized access points)
